# SeguroCali

Plataforma de alertas y prevención de violencia ciudadana para Cali, Colombia.

Proyecto desarrollado para **Crack the Code — Civic Tech**.

------

## 🎯 Misión

La misión de **SeguroCali** es empoderar a los ciudadanos de Cali mediante el uso de tecnología, permitiéndoles reportar situaciones de riesgo de forma anónima y en tiempo real, mientras se fortalece la comunicación con las autoridades. La plataforma busca contribuir a la reducción de la violencia, mejorar la prevención y generar comunidades más seguras a través de datos accesibles, visualizaciones claras y acción colectiva.

------

## 🚨 Problema

Cali cerró 2024 con más de 1.000 homicidios.
Esta app busca conectar ciudadanos con autoridades mediante reportes anónimos, mapas de calor y alertas en tiempo real.

------

## ⚙️ Stack tecnológico

- Next.js (App Router)
- React
- Canvas API (mapa de calor)
- (Próximamente) Supabase
